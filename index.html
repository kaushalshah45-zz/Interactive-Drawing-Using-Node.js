Interactive-drawing-using-node.js

This is an interactive drawing application where several users can collaborate and draw together. Whenever a user draws on the screen, all other users are updated in real time with the changes. There is also a chat feature where users can exchange messages in the side chat bar to co-ordinate and build something together. I have used socket.io to emit events to the Node.js server and express frameork to build the application.

For running this application:

  1. Install node.js. Install express using the command "npm install express". Install socket.io using the command "npm install socket.io".

  2. Extract the contents of this zip into any folder.

  3. Open a new Node.js command prompt or windows command prompt.

  4. Set path to the folder where you extracted the contents of this zip.

  5. Run the command "node index.js"

  6. Open any browser and go to "http://localhost:3000/"

  7. Enter your name.

  8. Open a new tab and go to "http://localhost:3000/"

  9. Again enter name

  10. Start drawing and see the results.
